POR QUE APENAS AS 4 PRIMEIRAS LETRAS SÃO SUFICIENTES?
O BIP39 foi projetado com uma restrição deliberada: nenhuma das 2048 palavras compartilha as mesmas 4 letras iniciais com outra palavra da lista.

Isso significa que ao digitar ou conferir uma seed phrase, basta identificar as 4 primeiras letras para determinar a palavra com 100% de certeza — as letras restantes são redundantes para fins de identificação.

Exemplos que ilustram o princípio:
 · POTAto  → apenas POTA já identifica esta palavra
 · POWEr   → apenas POWE já identifica esta palavra
 · PRACtice → apenas PRAC já identifica esta palavra

Essa propriedade reduz erros de digitação em carteiras de hardware e facilita a verificação manual — qualquer prefixo de 4 letras é uma chave unívoca para o índice binário correspondente.
